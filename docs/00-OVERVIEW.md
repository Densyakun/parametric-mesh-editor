# MeshNative Parametric Modeling System - System Design

## Vision

A new category of 3D modeling system that unifies:
- Blender's free mesh editing
- Fusion 360's history-based editing
- React's component architecture
- TypeScript's type safety
- Parametric CAD
- AI code generation

## Design Principles

1. **Mesh Native**: Internal representation is mesh, not B-Rep
2. **DAG Managed**: Dependency graph, not linear history
3. **TSX DSL**: Save format is human-readable TSX, not binary
4. **Plugin Equivalence**: Standard features and plugins share the same API
5. **AI First**: Designed for AI agent collaboration from day one
